# Ogani Ecommerce (Node.js + Express)

A full stack ecommerce web application built with Node.js, Express and MongoDB.

## Features

- User Registration & Login
- Product Listing
- Product Image Upload (Multer)
- Cart System
- Checkout System
- Order Management
- Admin Product CRUD
- SweetAlert Notifications
- Image Upload with Multer

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- EJS
- Bootstrap
- SweetAlert
- Multer


## Run Locally

1. Clone the repository

git clone https://github.com/AvinashParmar12/ogani-ecommerce-node.git

2. Install dependencies

npm install

3. Create `.env`

MONGO_URI=your_mongodb_connection
SESSION_SECRET=your_secret
PORT=5000

4. Start server

npm start
